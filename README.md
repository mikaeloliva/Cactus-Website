Cactus Website - README

Overview
The Cactus Website is a fully responsive front-end project showcasing ornamental plants through a clean, modern UI. It includes sections for browsing new plants, shopping, learning cactus care tips, and contacting the business. Built with HTML, CSS, and JavaScript, the site focuses on smooth navigation, visual clarity, and an elegant user experience.

Features
- Responsive layout optimized for all screen sizes
- Modern navigation bar with mobile menu toggle
- Home section with hero image and call-to-action buttons
- News section highlighting new plant arrivals
- Shop section with product cards and pricing
- Care section offering cactus and succulent care tips
- Contact section with social links, email, phone, and location
- Footer with company info and social media icons
- Smooth animations using ScrollReveal
- Clean, organized code structure with separate CSS and JS files

Tech Stack
HTML5 - Structure and content
CSS3 - Styling, layout, responsiveness
JavaScript (ES6) - Interactivity and animations
ScrollReveal.js - Scroll-based animations
Remix Icons - Iconography
